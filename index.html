Volunteer Surge

Volunteer Management Made Easy

Download this project as a .zip file Download this project as a tar.gz file

Recruit, schedule, and retain volunteers

Volunteer Surge is an open source volunteer management system written in Ruby on Rails. It allows volunteer shifts to be created by an administrater and for web users to sign up for each shift and share their signup on Facebook and Twitter.

Volunteer Surge is maintained by alexkroman
